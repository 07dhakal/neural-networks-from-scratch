# Neural Networks From Scratch

This repository contains scratch implementations of:

- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)
- Recurrent Neural Network (RNN)

All models are implemented using only NumPy, without deep learning libraries.

## Structure
- ann/ann_from_scratch.py
- cnn/cnn_from_scratch.py
- rnn/rnn_from_scratch.py
